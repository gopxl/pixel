## Pixel 2 Examples

This repository contains a few examples demonstrating [Pixel2](https://github.com/duysqubix/pixel2)'s functionality.

**To run an example**, navigate to its directory, then `go run` the `main.go` file. For example:

```
$ cd platformer
$ go run main.go
```
